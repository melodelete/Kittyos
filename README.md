# KittyOS

A whimsical, cat-themed web-based operating system simulation built with HTML, CSS, and JavaScript.

## Overview

KittyOS is a WebOS desktop environment. 

## Features

- **Cat-themed desktop interface** with custom cursors, and paw print effects
- Terminal with fun commands (meow, fact, paws, etc.)
- Notes app for creating and managing sticky notes
- Gallery for viewing images
- Clock with real-time display
- Browser for visiting websites
- Kitty Facts for learning interesting cat trivia
- Paw Gallery showcasing cute paw prints
- Draggable and resizable windows
- Start menu with application launcher
- System tray with clock, date, battery, and network indicators
- Desktop icons for quick app access
- Paw print effects when clicking anywhere on the screen

## How to Use

1. Simply open `index.html` in any modern web browser
